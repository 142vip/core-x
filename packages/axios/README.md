## 安装

```shell
# 安装
pnpm i @142vip/axios
```

## 功能

- [x] 封装`VipAxios`父类
- [x] `createAxiosInstance`函数
- [x] 提供默认拦截器函数，支持自定义拦截器

```ts
const axiosConfig = {}

// 创建对象
const vipAxios = new VipAxios(axiosConfig)
```
